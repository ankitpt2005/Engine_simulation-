# Engine Simulation (SI Engine)

Simple React-based simulator to study **mechanical efficiency** of a spark-ignition engine.

## Features

* RPM & Load control
* IP, BP, FP calculation
* Efficiency visualization
* Charts (Recharts)
* Basic simulation + observation

## Tech

* React (CRA)
* Three.js + React Three Fiber
* Recharts

## Run

```bash
npm install
npm start
```

## Formula

Mechanical Efficiency:
η = BP / IP
